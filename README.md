## unit-testing-jasmine-js
> use tools:
- npm v6.14.2
- node v12.6.0
- jasmine 3.5.0
- webpack v4.42.1

```sh
// install
$ npm install

// run buld(/dist)
$ npm run build

// http://localhosts:9000
---

// run dev
$ npm run watch
// run test
$ npm test
// http://localhost:9876/
```

test file: spec/index.spec.js
